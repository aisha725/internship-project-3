# STRYDE – Move Without Limits

STRYDE is a premium footwear e-commerce landing page developed as **Project 3** during my **Frontend Development Internship** at Decode Labs.

The goal of this project was to build a modern, responsive, and interactive brand website using only **HTML, CSS, and JavaScript**, focusing on clean design, smooth user experience, and frontend functionality without using any frameworks.

## Project Overview

This project represents a fictional footwear brand called **STRYDE** and is designed to provide a premium online shopping experience.

The website includes:
- A modern landing page design
- Interactive product browsing
- Product customization options
- Shopping cart functionality
- Theme switching (light/dark mode)
- Newsletter subscription form validation
- Responsive design for all devices

## Features

### Responsive User Interface
The website is fully responsive and works across:
- Desktop
- Tablet
- Mobile devices

### Light and Dark Theme Toggle
Users can switch between light and dark themes.  
The selected theme is stored using `localStorage`, so the preference remains saved after refreshing the page.

### Interactive Product Section
Users can:
- Browse different shoe categories
- Filter products by category
- View product pricing and badges
- Add products directly to the shopping cart

Categories include:
- Running
- Lifestyle
- Training
- Limited Edition

### Product Configurator
A custom product section allows users to:
- Select shoe size
- Choose a color
- Add the customized product to the cart

Validation ensures both options are selected before checkout.

### Shopping Cart
The cart system includes:
- Add/remove products
- Dynamic cart count
- Automatic subtotal calculation
- Slide-out cart drawer

### Newsletter Form Validation
The newsletter section validates:
- User name
- Email format
- User interests

After successful submission, a confirmation message is displayed.

### Custom UI Enhancements
Additional frontend features include:
- Custom animated cursor
- Smooth scrolling
- Scroll-triggered animations using `IntersectionObserver`
- Hover effects and transitions
